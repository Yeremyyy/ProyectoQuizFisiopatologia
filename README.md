# ProyectoQuizFisiopatologia
Este proyecto abarca la creacion de una pagina web, enfocada en Quiz de Fisiopatologia, Desarrollado con HTML, JavaScript, CSS. Aplicando MVC. Se hizo con toda la forma responsiva posible. Esta aplicacion permite la realizacion de Quiz y una tabla de puntaje para registrar los resultados del usuario. Se plantea futuras integraciones.
